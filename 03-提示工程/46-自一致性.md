# 第 46 题：自一致性（Self-Consistency）解码与推理质量提升

## 题目

什么是自一致性（Self-Consistency）解码？如何提升推理质量？

---

## 一、什么是 Self-Consistency？

**自一致性（Self-Consistency）** 指对**同一问题**用**采样**（非贪心）生成**多条**推理链与答案，再对**最终答案**做**多数投票**，取出现次数最多的答案作为最终输出。核心假设：正确的推理路径会多次出现，错误路径较分散；多数投票能过滤偶然错误、提升稳定与准确率。

---

## 二、如何提升推理质量？

- **与 CoT 结合**：先让模型用 CoT 生成多条“步骤+答案”，再只对**最终答案**做多数投票；既保留推理可解释性，又利用多路径降低单次错误。  
- **采样设置**：用一定 temperature 或 top-p 保证多样性（否则多条链会几乎相同）；通常 temperature 0.7 左右、生成 5–20 条再投票。  
- **适用任务**：数学、逻辑、选择题等有**离散标准答案**的任务效果最好；开放生成或长文本可用性有限（答案难以直接投票）。  
- **成本**：生成 N 条则调用成本与延迟约乘 N，需在效果与成本间权衡。

---

## 三、小结与面试要点

**小结**：Self-Consistency = 同题多次采样、对最终答案多数投票；常与 CoT 结合，用适度采样保证多样性，适合有离散答案的推理任务；能显著提升正确率但增加成本。

**面试要点**：

- 定义：同题多次采样，对最终答案多数投票。  
- 与 CoT：生成多条 CoT 链，对答案投票。  
- 适用：数学/逻辑/选择题；成本为 N 倍生成。

---

## 记忆要点

1. Self-Consistency = 同题多采样 → 对答案多数投票。  
2. 与 CoT 结合；适度 temperature 保证多样性。  
3. 适合离散答案任务；成本为 N 倍。
