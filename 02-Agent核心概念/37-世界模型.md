# 第 37 题：世界模型（World Model）在 Agent 中的作用

## 题目

什么是世界模型（World Model）？在 Agent 中的作用是什么？

---

## 一、什么是世界模型？

**世界模型（World Model）** 指对**环境与自身行为后果**的内部表示或预测模型：即“若我执行动作 A，环境会如何变化、会观察到什么”。它使 Agent 能在**不真实执行**的情况下，在内心“推演”可能的结果，从而支持**规划、反事实推理与样本高效学习**。

在经典控制与 RL 中，世界模型常指**状态转移与奖励的预测模型**（如学一个 $s_{t+1} = f(s_t, a_t)$）；在 LLM Agent 语境下，可泛化为：用模型（含 LLM）**预测或模拟**“某动作会导致什么观察/状态”，用于推理与规划。

---

## 二、在 Agent 中的作用

- **规划与前瞻**：在真正调用工具前，用世界模型预测“若调用工具 X 会得到什么结果”，从而比较不同动作、选择更优序列，减少试错。  
- **反事实与纠错**：当实际观察与预期不符时，用世界模型做“若当时选了另一动作会怎样”的推理，支持反思与策略改进。  
- **样本效率**：在 RL 或学习中，用世界模型生成“模拟轨迹”，减少对真实环境的依赖，提高数据效率。  
- **推理链中的“想象”**：CoT/ReAct 中的“假设先查 A 再查 B”的推理，可视为对世界模型的弱形式利用——用语言模型隐式预测“若这样做会得到什么”。

当前 LLM Agent 多数**没有显式、独立的世界模型**，而是依赖 LLM 的隐式“常识与因果”做一步接一步的推理与工具调用；显式世界模型（学环境动力学、或用 LLM 做状态/结果预测）是研究热点，用于更稳的规划与更少真实调用。

---

## 三、小结与面试要点

**小结**：世界模型 = 对“动作→环境变化/观察”的预测或表示；在 Agent 中支持规划与前瞻、反事实推理、样本效率与推理中的“想象”；当前多数 LLM Agent 依赖隐式推理，显式世界模型是研究前沿。

**面试要点**：

- 定义：对动作导致的环境/观察结果的预测或内部表示。  
- 作用：规划与前瞻、反事实推理、样本效率、推理中的“想象”。  
- 现状：LLM Agent 多依赖隐式推理；显式世界模型用于更强规划与样本效率。

---

## 记忆要点

1. 世界模型 = 预测“动作→环境/观察”的模型或表示。  
2. 作用：规划与前瞻、反事实、样本效率、推理中的想象。  
3. 当前 Agent 多无显式世界模型；显式世界模型是研究热点。
