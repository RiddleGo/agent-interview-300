# 知识卡片风格与规格

当前采用：**优雅复古风格（Elegant Vintage）**，详见下文「02」；孟菲斯版保留于 `cards-showcase.html` 的图片展示。

---

## 01 | 孟菲斯解构主义（Memphis Deconstructed）

### 设计哲学
打破常规秩序，创造有意识的视觉冲突与趣味张力。

### 色彩系统
- 高饱和不协调组合：**亮粉 + 青绿 + 柠檬黄 + 橙色**
- 辅以 **黑白灰 / 金属色** 平衡

### 字体排版
- 几何感强烈的无衬线字体
- 文字刻意 **倾斜、错位、重叠、切割**，打破网格系统

### 图形元素
- 解构重组的几何形状、波点、条纹、锯齿
- 手绘线条、涂鸦、色块、箭头

### 肌理质感
- 拼贴感、剪贴报手法、故障艺术效果

### 参考源流
埃托雷・索特萨斯作品、解构主义设计、后现代艺术、实验性排版。

### 适用场景
艺术、设计、时尚、创意文化主题。

### 技术要点
- CSS grid 打破式布局
- `transform` 多重变形
- `mix-blend-mode` 色彩混合

## 02 | 优雅复古风格（Elegant Vintage）※ 当前

### 设计哲学
重现 20 世纪初期印刷品的精致美学，典雅、成熟且历经时间考验。

### 色彩系统
- **纸张**：米色/淡黄纸张质感（#f5f0e6）
- **墨色**：深棕、暗红系（#2c2420、#5c524a）
- **点缀**：古典棕（#8b4513）、淡金分隔

### 字体排版
- 衬线字体：Noto Serif SC、Cormorant Garamond
- 对称庄重，传统书籍式留白与层次
- 日期/小注使用细体与斜体

### 必备元素（每张卡片）
- **日期区域**：顶部小号字、字间距加宽、大写/缩写
- **标题与副标题**：主标题加粗，副标题斜体+下分隔线
- **引用区块**：左侧竖线+浅棕底，公式或金句
- **核心要点列表**：破折号引导，可带小标签（步骤/复杂度/要点等）
- **编辑笔记/小贴士**：虚线框边栏，斜体，编辑提示或面试一句话

### 装饰与质感
- 顶部细线渐变装饰
- 极淡纸张纹理（SVG 噪点）
- 细边框与柔和投影

### 参考源流
The New Yorker、老式法国时尚杂志、古典书籍设计。

## 尺寸规格

| 项目 | 说明 |
|------|------|
| **基准尺寸** | 宽度 400px，高度不少于 1280px |
| **比例关系** | 遵循 3:10 的垂直长卡片比例，保持竖版布局 |
| **显示方式** | 卡片自左向右平铺展示，保持一致的顶部对齐 |
| **响应适配** | 主体内容在固定宽度内，确保在不同设备上保持竖版体验 |
