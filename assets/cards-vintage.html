01 | Self-Attention（自注意力）

序列中每个位置根据与其它位置的相似度，对 Value 加权求和。

Attention(Q,K,V) = softmax(QKT / √dk) V
  • 步骤 Q,K,V → 分数 QKT → 缩放+softmax → AV
  • 复杂度 时间 O(n²d)，空间 O(n²)
  • 要点 长距离依赖、对序列顺序不敏感、缩放防梯度消失、多头增强表达
编辑笔记 长序列时 n² 主导显存与算力，可考虑 Sparse / Local Attention 或线性注意力。

02 | 位置编码（Positional Encoding）

为 Transformer 注入顺序信息，否则 Self-Attention 无法区分先后。

绝对：每位置 t 一向量 pt；相对：在 Attention 中编码 i 与 j 的差值 i−j。
  • 绝对 Sinusoidal、可学习；优点简单，缺点外推差
  • 相对 T5、RoPE、ALiBi；优点外推好，缺点实现稍复杂
  • 选用 Decoder-only 常用 RoPE/ALiBi；Encoder 常用绝对或 T5 相对
小贴士 面试可一句话区分：绝对回答「第几个位置」，相对回答「相距多远」。
March 2025 · LLM 基础

03 | Layer Normalization

对单样本、层内特征标准化，不依赖 batch 与序列长度。

LN(h) = γ · (h−μ)/√(σ²+ε) + β，μ、σ 在特征维 d 上求。
  • 与 BN BN 对 batch 维；LN 对特征维，与 n、batch 无关
  • 选用 Transformer 用 LN：序列变长、小 batch、训练/推理一致
编辑笔记 序列模型用 LN 是标配；BN 在 NLP 中多用于小固定长度或非自回归任务。
March 2025 · LLM 基础

04 | 注意力掩码（Attention Mask）

在 softmax 前屏蔽某些位置，使对应注意力权重为 0。

scores + M；M=−∞ 处 softmax 后为 0。
  • Padding mask 屏蔽 pad；有 pad 即用
  • Causal mask 屏蔽 j>i；自回归训练与推理必用
小贴士 两种可叠加：先 causal 再在 pad 位置加 padding mask。
March 2025 · LLM 基础

05 | KV Cache

缓存已算过的 K、V，每步只算新 token 的 kt, vt 并拼到缓存。

K1:t = [K1:t−1; kt]，V 同理；Attention(qt, K1:t, V1:t)。
  • 作用 避免对历史重复算 K/V，降低解码计算与延迟
  • 代价 显存随序列长度、层数、batch 线性增
编辑笔记 长序列需长度/批大小控制或 PagedAttention 等优化。
March 2025 · LLM 基础

06 | 解码策略对比

Greedy / Beam / Temperature / Top-p 的取舍。

开放生成多用 Temperature + Top-p；翻译/摘要可用 Beam；要确定性用 Greedy 或低 T。
  • Greedy 每步最大概；简单快，单调
  • Beam 保留 k 条；连贯可控，贵
  • T / Top-p 调多样性；常合用
小贴士 创意对话：T+Top-p；需可复现：低 T 或 Greedy。
March 2025 · LLM 基础

07 | 困惑度（Perplexity）

测试集上交叉熵的指数，越低表示语言建模越好。

PPL = exp(−1/N Σ ln P(xi|x<i))。
  • 含义 平均“等价选项数”；PPL=1 理想
  • 用途 比较模型与 baseline；主要反映语言建模
编辑笔记 与下游任务相关但不完全一致，需结合下游指标。
March 2025 · LLM 基础

08 | 涌现能力（Emergent Abilities）

随规模增大在较窄区间内突然出现或跃升的能力。

少样本、CoT、指令遵循、工具使用等。
  • 定义 难以由小模型外推预测
  • 成因 Scaling、评估曲线非线性、数据与目标
小贴士 并非“魔法”，与评估与 scaling 行为有关。
March 2025 · LLM 基础

09 | 上下文学习（ICL）

不更新参数，仅在 prompt 中给示例，模型即能按任务预测。

示例1→输出1；示例2→输出2；查询 → 模型生成。
  • 与微调 微调改参数；ICL 不改，靠前向表示与注意力
  • 机制 隐式推断、注意力类比、任务方向等（无定论）
编辑笔记 大模型上效果显著，小模型有限。
March 2025 · LLM 基础

10 | Chain-of-Thought（CoT）

先写推理步骤、再写答案；任务分解、格式对齐。

“Let's think step by step” 或 few-shot 带步骤示例。
  • 有效性 规模依赖、降低单步难度、激活推理模式
  • 增强 Self-Consistency：多条链取多数答案
小贴士 数学/逻辑题优先考虑 CoT + Self-Consistency。
March 2025 · LLM 基础

11 | Encoder / Decoder 架构对比

理解型 vs 生成型 vs 源-目标型。

Encoder-only 理解；Decoder-only 生成+ICL；Encoder-Decoder 翻译/摘要。
  • Encoder-only BERT；分类、NLI、抽取、检索
  • Decoder-only GPT/LLaMA；对话、通用
  • Enc-Dec T5/BART；有明确源-目标
编辑笔记 大模型主流为 Decoder-only。
March 2025 · LLM 基础

12 | 模型 / 数据 / 流水线并行

DP 同模型不同数据；MP/TP 拆层；PP 按层分段流水。

大模型训练常 DP + TP + PP 组合。
  • 数据并行 单卡能放下模型时首选
  • 模型/张量并行 单卡放不下层时用
  • 流水线并行 层多、整模型放不下；注意气泡
小贴士 Megatron-LM 即 TP+PP+DP。
March 2025 · LLM 基础

13 | 混合精度训练

前向/反向 FP16，主权重+优化器 FP32；loss scaling + 敏感层 FP32。

算力与带宽更优；FP32 防累积误差；scaling 防梯度下溢。
  • 优势 加速、省显存、效果与 FP32 相当
编辑笔记 V100/A100 上 FP16 有 Tensor Core 加速。
March 2025 · LLM 基础

14 | 梯度累积

多步小 batch 梯度累加（或平均），每 k 步更新一次。

有效 batch = 小 batch × k；单步显存由小 batch 决定。
  • 作用 缓解 OOM，等效大 batch 训练
  • 注意 BN 统计仍按小 batch；k 大时调学习率
小贴士 无 BN 的 Transformer 用梯度累积很自然。
March 2025 · LLM 基础

15 | LoRA

W′ = W + AB，A、B 低秩；只训 A、B，冻结 W。

参数量 r(d+k) ≪ dk；A 小随机、B 零初始化。
  • 放置 常对 attention Q/V 与 FFN 加 LoRA
  • 推理 可合并为 W′ 或保留多组 A,B
编辑笔记 秩 r=8/16/32 常用。
March 2025 · LLM 基础

16 | 微调方法对比

Full FT / LoRA / Adapter / Prompt Tuning。

LoRA 最常用；Adapter 多任务；Prompt Tuning 极省参数。
  • Full FT 全参数，显存高
  • LoRA 省显存、可合并推理
  • Adapter 模块化、多任务
小贴士 资源紧时优先 LoRA 或 Prompt Tuning。
March 2025 · LLM 基础

17 | 模型量化

权重/激活从 FP 映射到低比特整数，省显存、提吞吐。

INT8 对称 per-tensor/channel；INT4 分组+GPTQ/AWQ 校准。
  • INT8 精度损失小、硬件支持好
  • INT4 更省显存，依赖校准与组大小
编辑笔记 QLoRA 即 LoRA + INT4 量化 backbone。
March 2025 · LLM 基础

18 | GQA 与 MQA

MQA 所有头共享 K、V；GQA 组内共享，折中。

动机：解码时省 KV Cache、提吞吐。
  • MQA KV Cache 最小，表达弱
  • GQA 省 Cache 且保留多数表达；LLaMA-2 采用
小贴士 大模型解码端 GQA 已成主流。
March 2025 · LLM 基础

19 | RoPE

旋转位置编码：Q、K 按位置旋转，score 只依赖 i−j。

相对性、外推好、不占嵌入；LLaMA 等广泛采用。
  • 优势 利于语法/指代，外推友好
编辑笔记 Decoder-only 大模型主流位置编码。
March 2025 · LLM 基础

20 | SwiGLU

SwiGLU(x) = Swish(W1x) ⊙ (W2x)；门控 + Swish 平滑。

PaLM/LLaMA 将 FFN 换为 SwiGLU，效果优于 ReLU/GELU。
  • 代价 多一路线性，需在隐藏维上平衡参数量
小贴士 大模型 FFN 常见选择。
